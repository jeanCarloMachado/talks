Inicializando o projeto
-----------------------

```
$ pip install mkdocs
$ mkdocs new git-docs
$ cd git-docs
$ mkdocs serve
$ browser http://localhost:8000
```

Todas as operações vão acontecer dentro desse diretório (git-docs).
